# DMF Academic Analytics

> Enterprise Learning Analytics Platform for Thai Primary Schools

DMF Academic Analytics is an enterprise-grade Learning Analytics Platform designed to analyze student assessment data and provide actionable insights for school administrators, teachers, and educational organizations.

---

## 🎯 Vision

Transform assessment data into meaningful learning insights that help improve student achievement through data-driven decision making.

---

## ✨ Features

### Current (v0.1)

- Project Foundation
- Configuration Management
- Environment Loader
- Documentation Suite
- Architecture Governance
- Quality Gate Workflow

### Planned

- O-NET Analytics
- NT Analytics
- RT Analytics
- Pre-Test / Post-Test Analytics
- Student Growth Analytics
- Learning Standard Mastery
- Executive Dashboard
- Teacher Dashboard
- AI Recommendation Engine
- Learning Evidence Analytics

---

## 🏗 Architecture

The project follows a **Modular Monolith Architecture**.

Core principles include:

- Single Source of Truth
- Modular Design
- Domain-Driven Structure
- RFC → ADR → IDR Governance
- Quality Gate Workflow
